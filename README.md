# Mall Section Dashboard
## Project Idea
This project is a web application dashboard that allows users to explore different sections inside a mall such as restaurants, clothing stores, games, and gym facilities.

## Business Problem
Visitors in large shopping malls often struggle to find information about available services such as restaurants, clothing stores, entertainment areas, and gym facilities. This information is usually not organized in one place, which makes it difficult for visitors to locate services easily.

## Proposed Solution
The Mall Section Dashboard provides a centralized platform where users can browse mall sections and view useful information about each service in an organized way.

## Application Sections

### Restaurants
- name
- type
- opening hours
- floor
  
### Clothing
- name
- category
- floor

### Arcade
- name
- type
- price
- floor

### Gym
- service name
- type
- floor
